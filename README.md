# Client

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.1.3.

## Clone o Projeto

bash
```
git clone https://github.com/Santana-Fernando/Client_angular.git
```

## Instale os pacotes

Na pasta Client_angular, rode
```
npm i
```

## Rode o projeto

```
npm start
```
